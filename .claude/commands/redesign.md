# Redesign Component

Redesign an existing component to make it more modern, beautiful, and polished.

## Instructions

Read `$ARGUMENTS` and redesign the component with these principles:

**Visual upgrade checklist:**
- [ ] Replace any `rounded-md` with `rounded-xl` / `rounded-2xl`
- [ ] Add `hover:shadow-md transition-shadow` to cards
- [ ] Improve typography hierarchy — bigger contrast between heading and body sizes
- [ ] Add micro-interactions to all interactive elements
- [ ] Replace hardcoded colors with CSS variables (`text-primary`, `bg-primary-soft`, etc.)
- [ ] Add subtle gradients or background tints to section separators
- [ ] Ensure spacing is generous — minimum `p-4` in cards, `gap-4` in grids
- [ ] Add `group` and `group-hover:` effects for card interactions
- [ ] Verify image containers use proper `aspect-ratio` + `object-cover`

**Reference designs:** Linear, Vercel, Luma.events, Stripe

Show before/after side by side as comments in the code, then write the full redesigned component.
