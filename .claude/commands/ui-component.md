# UI Component Generator

Generate a beautiful, production-ready UI component following the 6sotok design system.

## Instructions

The user wants: `$ARGUMENTS`

Create a React/TypeScript component using Tailwind CSS that:

1. Follows the 6sotok design tokens (primary: #066F36, Inter font, zinc neutrals)
2. Uses `rounded-2xl` / `rounded-xl` border radius — never `rounded-md`
3. Includes hover states and `transition-all duration-200` on interactive elements
4. Is mobile-first responsive
5. Uses Lucide React for any icons
6. Looks like it belongs on Linear.app or Vercel.com

**Design standard check before writing:**
- What is the visual hierarchy?
- Where does the eye go first?
- Is whitespace generous enough?
- Is primary color used sparingly?

Output the complete component code, then describe the key design decisions made.
